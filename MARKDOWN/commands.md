## To start server:

- `cd backend` -> `npm run start`

## To start frontend:

- `cd frontend` -> `npm run dev`

## To push changes:

- `git add .` \*OR\* `git add [folder name]`
- `git commit -m "your commit message"`
- `git push origin main`
